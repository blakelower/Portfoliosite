About

Process

Pre-Process

Collect Information, Personas, Set up Goals

Early Stage

Build Figma for client, check in with them to see if the mock up is what they were looking for

Start design

Depending on the clients choice of CMS I would start designing by this stage. I would also begin a mood board to fit a certain aesthetic.

Test

Check in with client, make sure everything is responsive on certain devices. I would also make sure that each image has an alt tag, and that the SEO is set up correctly.

Deploy

Deploy the site to Netlify, or clients choice such as Wordpress or Webflow.
